
=================================
Welcome to Curie's documentation!
=================================

.. only:: html

   Curie is a python toolkit to aid in the analysis of experimental nuclear
   data, with primary applications in gamma-ray activation analysis and the
   charged-particle stacked-target activation technique.  The
   :ref:`intro` page gives an overview of its features.

   This documentation is also available as a single PDF: `Curie.pdf <Curie.pdf>`_.
