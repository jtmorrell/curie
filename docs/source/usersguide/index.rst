============
User's Guide
============

Welcome to the Curie User's Guide!  The user's guide is an introduction to the essential classes in Curie, along with some examples.  For more details about the methods and attributes of these classes, visit the Curie :ref:`api`.

The :ref:`getting_started` page maps out the classes — grouped by what you would use them for — and gives a few examples to run.  The other links provide more specific details and worked examples for each topic.
